fuzzy_ratio(field_names, text, allow_partial=False, max_partial_difference=100)[source]¶

Given a snippet of text, computes the fuzzy ratio between the text and text that is stored on one or more fields on the object.

Parameters

  • field_names – The names of the fields to compare the text against.

  • text – A string of text to compare

  • allow_partial – Whether or not to allow partial fuzzy ratios when computing text similarity.

  • max_partial_difference – The maximum difference between the absolute and partial ratio that’s

allowed to return a result. :return: The fuzzy ratio between the object and the comparison text

Parameters

  • allow_partial – Whether or not to allow partial fuzzy ratios when computing text similarity.
